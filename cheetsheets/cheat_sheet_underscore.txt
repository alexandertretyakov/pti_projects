Коллекции
    each - служит для перебора всех элементов коллекции и применения функции к каждому из элементов.
    map - получает на вход массив и возвращает новый массив, созданный путем преобразования
          элементов оригинальной коллекции.
    reduce - выполняет “склеивание” элементов коллекции - все значения элементов будут
             объединены в одно значение.
    reduceRight - тоже, что и reduce, только обход начнётся с конца коллекции.
    find - вернёт первое значение, прошедшее проверку на истинность.
    filter - вернёт массив всех значений, прошедших проверку на истинность.
    where - вернёт массив из входящих элементов, для которых совпадают значения соответсвующих ключей.
    findWhere - сработает аналогично where, только вернёт первый элемент, соответствующий критериям.
    reject - возвращает массив, содержащий все элементы, которые проходят тест на истинность.
             Противоположность filter.
    every - вернёт true, если все значения в списке проходят тест на истинность.
    some - вернёт true, если любое из значений в списке проходит тест на истинность.
    contains - вернёт true, если содержится искомое значение.
    pluck -  вернёт массив состоящий из значений по указанному ключу.
    max - вернёт максимальное значение в списке.
    min - вернёт минимальное значение в списке.
    sortBy - сортировка по возростанию/убыванию.
    shuffle - вернёт перетасованную копию массива.
    sample - возвращает массив с n-случайных элементов, если n не был передан, вернёт массив с
             одним случайным элементом.
    toArray - сконвертирует псевдомассив в массив, удобен, например, для преобразования
              объекта arguments.
    size - вернёт кол-во элементов в массиве (как length).
    partition - разобъёт массив на два вложенных массива, в один пойдут значения прошедшие
                проверку ф-ции predicate, а в другой все остальные.

Массивы
    first - возвращает первый элемент массива.
    last - возвращает последний элемент массива.
    rest - возвращает массив всех элементов, кроме первого или отсекает любое указанное
           кол-во элементов.
    compact - возвращает копию списка с удаленными всеми ложными значениями.
    flatten - выравнивает вложенные массивы (любого уровня вложенности), возвращая склеенный
              одноуровневый массив.
    without - возвращает копию массива со всеми удаленными запрашиваемыми значениями.
    union - вернёт массив, объеденяя все уникальные элементы проброшенных массивов.
    intersection - верёт массив значений, которые повторяются в каждом из входящих массивов.
    difference - аналогичен without, с той разницей, что значения, которые необходимо исключить
                 передаются в виде массивов
    uniq - вернёт массив состоящий из уникальных элементов. Повторяющийся элемент вернёт один раз.
    indexOf - вернёт позицию искомого элемента в массиве.
    lastIndexOf - тоже, что и indexOf, только поиск элемента начнётся с конца массива.
    findIndex - вернёт первый индекс элемента, для которого ф-ция предикат вернёт true.
    findLastIndex - аналогично findIndex, только обход массива начнётся с конца.

Функции
    bind - связывает ф-цию с объектом. При вызове ф-ции, this, будет указывать на объект
    throttle - вернёт ф-цию, которая, при повторных вызовах исполнится не чаще одного раза в
               указанный промежуток времени. Полезна при использовании обработчика событий,
               который вызывается слишком часто.
    once - венёт ф-цию, которая может быть вызвана всего 1 раз. Все остальные её вызовы будут
           возвращать значение, вычисленное в первый раз.
    compose - вернёт ф-цию, создавая цепной метод, в котором возвращаемые значения каждой ф-ции
              используются как аргументы другой ф-ции. Говоря проще объединение
              ф-ций f(), g() и h() даёт f(g(h())).

Объекты
    keys - вернёт массив ключей объекта.
    values - вернёт массив значений объекта.
    pairs- вернёт массив, в котором вложенные массивы содержат пары объекта.
    invert - вернёт копию входящего объекта, где ключи - значения, а значения - ключи. Чтобы это
             работало, все значения должны быть уникальными.
    create - создаст новый объект, с заданным прототипом.
    extend - скопирует все свойства из объектов в целевой объект. Если целевой объект имеет
             одноимённые свойства, то значение каждого будет перезатёрто значениями из следующего
             объекта.
    pick - возвращает копию объекта, содержащую только указанные ключи.
    omit - возвращает копию объекта, без указанных ключей.
    defaults - наподобии extend, только свойства целевого объекта, которые уже определены,
               перезатираться не будут.
    has - проверяет, содержит ли объект искомый ключ.
    isEqual - выполнит оптимизированное "глубокое" сравнение двух объектов.
    isMatch - проверит, содержаться ли в объекте искомые пары ключ/значение.
    isEmpty - вернёт true, если объект пустой.
    isArray - вернёт true, если пробрасывается массив.
    isObject - вернёт true, если пробрасывается объект.
    isFunction - вернёт true, если пробрасывается функция.
    isString - вернёт true, если пробрасывается строка.
    isNumber - вернёт true, если пробрасывается число.
    isBoolean - вернёт true, если пробрасывается Булевое значение.
    isNull - вернёт true, если пробрасывается Null.
    isUndefined - вернёт true, если пробрасывается undefined.

Утилиты
    noConflict - отвяжет переменную _ от объекта Underscore.
    noop - всегда возвращает undefined.
    random - возвращает случайное целое число, в диапазоне от и до включительно. Если передаётся
             один аргумент, будет использоваться диапазон от 0 до указанного значения.
    uniqueId - генерирует абсолютно унакальный ID для модели или элемента DOM, на клиентской
               стороне. Если будет передан префикс (аргумент), ID будет добавлен к нему.
    result
    now - возвращает кол-во миллисекунд начиная с 1 января 1970г.
